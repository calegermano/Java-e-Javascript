Possibilidades com uso de JavaScript

Java Script e Java são a mesma coisa?

Java é uma linguagem de programação e plataforma de computação liberada pela primeira vez pela Sun Microsystems em 1995, já o JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.

Quando surigu o Java Script?

O JavaScript foi criado em 4 de dezembro de 1995.

Quem foi o responsável pela criação do Java Script?

Foi criado por Brendan Eich, a pedido da empresa Netscape.

Qual o caminho trilhado pelo reponsável pela criação do Java Script, trata-se de alguma empresa ainda no mercado?

Brendan Eich trabalhava na Netscape Communications, o objetivo era criar uma linguagem que rodasse no navegador e facilitasse a interação com páginas web. A Netscape não existe mais como empresa, mas o JavaScript evoluiu e hoje é mantido pela organização Ecma International (através do comitê TC39), que define seus padrões.
